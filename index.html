수십 개의 구인공고를 일일이 열어보지 않아도, 기업들이 요구하는 기술과 자격증 트렌드를 빠르게 파악할 수 있습니다.

IT / 개발자 디자인 마케팅 전기 / 전자 기계 / 제조

주요 기능

🔍

직군별 기술 순위

선택한 직군에서 기업들이 가장 많이 요구하는 기술 스택을 순위로 보여줍니다.

🏆

자격증 트렌드

직군별로 우대받는 자격증이 무엇인지, 몇 개의 기업이 요구하는지 확인할 수 있습니다.

📈

데이터 시각화

기술과 자격증을 차트로 시각화하여 직관적으로 트렌드를 파악할 수 있습니다.

서비스 구조

1

채용공고 수집

API를 통해 선택한 직군의 채용공고 데이터를 수집합니다.

2

AI 키워드 추출

OpenAI API를 활용하여 공고 텍스트에서 기술과 자격증을 자동으로 분류합니다.

3

데이터 저장 및 집계

추출된 키워드를 MySQL에 저장하고 빈도수를 집계합니다.

4

시각화

기술 순위와 자격증 순위를 차트로 시각화하여 사용자에게 제공합니다.

기술 스택

Python OpenAI API MySQL JavaScript Chart.js Selenium Flask
